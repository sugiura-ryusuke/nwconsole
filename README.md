# nwconsole
Network Console for Embedded Software

組み込みソフトウェア向けの開発用ツールです。ターゲット機器を TCP サーバーとして、開発 PC 上で本プログラムを TCP クライアントとして動作させる構成を想定しています。双方向の通信に対応しており、ターゲット機器からデバッグログを送信して開発 PC のコンソールに表示しつつ、開発 PC のコンソールからデバッグコマンドを送信してターゲット機器の制御を行うといった使い方が可能です。

また、本プログラムは TCP と同一のポートを使用する UDP サーバーとしても動作し、UDP で受信したメッセージはそのまま TCP サーバーに転送されます。本プログラムを常駐プロセスとして起動させておき、スクリプトから UDP を使ってメッセージを送信することで、本プログラムを経由してターゲット機器にデバッグコマンドを送り込む、といった使い方ができます。
